# strava-scrapper
strava scrapper with requests and bs4

Installation via PyPi:
-------------

```
$ pip install .
```

Usage:
-------------

```
$ python commands.py --help
```

# Docker installation

build image:
-------------

```
$ docker build -t scrava-scrapper --rm .
```

run container:
-------------

```
$ docker run -it --name scrava --rm scrava-scrapper
```
